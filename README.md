# Frevora

`Free`(自由) + `Real`(实时) + `Vision`(视觉) + `Atom`(原子)
这个项目尝试基于32位单片机，将`FreeRTOS`和`LVGL`有机结合，打造出便于开发与移植且具有市场前景的嵌入式平台。
